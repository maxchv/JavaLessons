Создать приложение для добавления товаров в базу данных.
Каждый продукт имеет следующие свойства:
	id
	Название
	Описание
	Цена
	Группа


Группа товаров это:
	Бытовая техника
	Ноутбуки и компьютеры
	Смартфоры
	Товары для дома
	Инструменты и автотовары

Добавление товаров происходит на одной странице, при этом
при нажатии на кнопку "Добавить" данные товара сохраняются
в текущей сессии.

Таким образом можно добавить несколько товаров.

При переходе на страницу "Все добавленные товары" отображается
список с добавленными товарами (из сессии).
Нажатие на кнопку "Сохранить в б/д" позволяет сохранить данные
в базе данных, при этом товары из сессии удаляются.
