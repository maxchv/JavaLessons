Задание:
--------

Создать класс User. Объявить поля:
login, password, а также необходимые методы и другие поля.

В классе User создать два внутренних класса LoginInfo и 
Preferences

В классе LoginInfo  определить поля содержащие данные о логине,
методы для авторизации и выхода.

В классе Preferences объявить дополнительные поля такие как name,
dateOfBirth, email, facebook, instagram и т.д.

Продемонстрировать применение класса User: создать пользователя с
заданными логином и паролем, авторизироваться, вывести дополнительную
информацию о пользователе.

Дополнительное задание:
-----------------------

Создать класс Array со статическим методом sort.
Метод sort принимает массив Objесt и объект Compare
Класс Compare содержит один абстрактный метод cmp принимающий два
Object. Назначение метода - сравнить два аргумента и вернуть одно из значений 1, -1, 0.

Создать массив объектов Object[] arr = new Object[]{6, 9, 1, 6, 9};

Применить метод sort класса Array для сортировки массива arr передав в качестве второго 
аргумента экземпляр анонимного класса реализующий класс Compare.
