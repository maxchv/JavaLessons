Написать web приложение с возможностью регистрации пользователя.
При регистрации пользователь вводит: логин, пароль и email.
Для подтверждения регистрации пользователю высылается сообщение
на адрес указанный email.
В письме необходимо указать url при переходе на который регистрация
пользователя будет подтверждена.
При формировании url можно использовать JSESSIONID пользователя 
при регистрации.
После подтвержения регистрации пользователь может успешно войти на сайт.

Предусмотреть перевод интерфейса на английский, украинский и русский языки.
Язык определеяется исходя из локали запроса.
