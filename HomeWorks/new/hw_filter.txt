Задание:

Создать фильтр AuthenticationFilter для проверки аутентификации польователя.
Признаком того, что пользователь вошел будет наличие кука authentication.
Если кука нет, то в фильтре производится перенаправление на сервлет LoginServlet.
Данные пользователей сохранять в базе данных.

Дополнительное задание: 

Написать фильтр LoggerFilter для журналирования запросов ко всем сервлетам.
В журнал необходимо сохранять следующие данные: время запроса, url запроса,
метод запроса.
Журнал хранить в атрибуте приложения (request.getServletContext().setAttribute("logger", ...))
Для отображения журнала создать LoggerServlet посредством которого отобразить данные в виде таблицы.
