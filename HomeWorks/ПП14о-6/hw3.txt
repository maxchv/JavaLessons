Задание 1
Реализовать односвязный список в виде класса List (не использовать стандартные интерфейсы и списки).

В класс List должен входить вложенный (inner) класс Data
представляющий отдельный узел списка.

В классе List реализовать методы добавления элемента в список (add),
извлечение последнего элемента списка (pop), преобразования всего элемента
списка в строку (toString) 

Привести пример применения списка для хранения списка автомобилей (из предыдущих д.з.)

Задание 2.
Реализовать класс ListException, EmptyListException и FullListException

ListException является наследником класса Exception
EmptyListException наследуюет класс ListException
FullListException также наследуюет класс ListException

В классе List добавить исключительную ситуацию в метод:
	
	- removeLast() на тот случай, если производится попытка удаления элемента
	  из пустого массива obj (исключение типа EmptyListException)
	  
	- add() на тот случай, если массив obj полностью заполнен данными  (исключение типа FullListException)  

Задание 3.
Написать обобщенный класс Map<K, V> хранящий список данных в ввиде ключ (тип K), 
значение (тип V). Не использовать стандартные классы и интерфейсы!!!
Map содержит массив экземпляров класса Pair<K, V>.

В Map реализовать методы:
	- добавления add(K key, V value)
	- удаления по ключу remove(K key)
	- получения по ключу get(K key)
	- преобразование к строке toString() 
	
На базе созданного класса создать англо-русский словарь.
Привести пример работы со словарем
